## Sentence Starters / Satzanf√§nge

Use these to explain code in English:

1. **"The condition checks if..."**
   - The condition checks if the age is greater than 13.

2. **"If the condition is True, then..."**
   - If the condition is True, then the message is printed.

3. **"This code will print..."**
   - This code will print "You are a teenager!"

4. **"When the value is..., the output is..."**
   - When the value is 25, the output is "It's warm!"

5. **"In this example..."**
   - In this example, we check the temperature.
