## ğŸ“ Teaching Tips

### Before the Lesson:

1. **Preview the slides** in fullscreen mode
2. **Test the timer** to understand how it works
3. **Bookmark key slides** (you can note slide numbers)
4. **Have backup** - print a few key slides just in case

### During the Lesson:

1. **Use fullscreen** for professional look
2. **Set timer** at start of each phase (Press T)
3. **Navigate smoothly** with arrow keys
4. **Check notes** if you forget timing (Press N)
5. **Don't rush** - slides match your lesson plan timing

### Phase-by-Phase Guide:

**Phase 1 (Slides 3-4):**
- Spend 4-5 minutes on slide 3
- Use slide 4 to show student responses
- Transition naturally to grammar analysis

**Phase 2 (Slides 5-9):**
- Slides 5-7: Grammar focus (6 minutes)
- Slides 8-9: Code connection (4 minutes)
- Don't skip slide 6 - the rule is essential!

**Phase 3 (Slides 10-11):**
- Show slide 10 for instructions
- Students work on worksheet (not on slides)
- Use slide 11 for common mistakes review

**Phase 4 (Slides 12-13):**
- Slide 12 for task overview
- Slide 13 for struggling students
- Students write in notebooks/worksheets

**Phase 5 (Slides 14-18):**
- Move through these relatively quickly
- Pause on quiz slides for discussion
- End with homework slide
